# 程序设计及应用（Python）

与计算机信息技术同理，学部用来教Python编程的课。

Python相比C++更常用，因此实用性和扩展性更高。建议在课堂学习基础上，看一些如

- CS61A: Structure and Interpretation of Computer Programs (UC Berkeley) 
- CS50P Introduction to Programming with Python (Harvard University)
- MIT6.100L: Introduction to CS and Programming using Python (MIT) 

之类的课程，加深对Python的理解。

正常过程化考试不是很难，正常学就能过。最后期末考试苏州大学用的是纸笔考试（逆天），因此平时要多手打指令而非Tab补全。

给我上课的是甄田田老师，平常水准，但对待学生很厚道，经常在考前无私分享往年题库让大家心理有个准备。
