# 苏州大学材料与化学化工学部完全指南

这是一篇苏州大学材化部的本科学习指南，也是对自己大学三年学习生涯的一个纪念。

本指南由苏大材化22级学长编写并维护，可以帮助学弟学妹对材化部快速了解，高效完成材化部本科专业的学习。

本指南目前包括以下部分（如有补充与建议欢迎评论或私信）：

- 苏大材化课程：本节含苏州大学材料与化学化工学部从大一到大四各类课程的学习资料，包括我自己的试卷及复习笔记，以及材化部历代学长学姐流传下来的课程资料。

- 苏大材化资料：本节以活动的形式，涵盖了在苏大材化求学可能遇到的各种事情的指南，并附有相关的建议。

- 苏大材化书籍推荐：本节中涵盖了一些在材化学习时应读的好书，可以加深对课内课外知识的理解。

- 苏大材化学习时光：本节中以回忆录的形式，记载了我在苏大求学的过程。
